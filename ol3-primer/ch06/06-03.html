地图联动

OpenLayers 3采用了MVC模型，V对应的就是View，这样Map和它就是分离开来，这样做有什么好处，那么就让我们拭目以待：

地图1

地图2

拖动地图1的同时，看看地图2有什么变化，是不是很神奇，两个地图都在一起动，这是如何做到的呢？

<script>
    // 创建一个视图
    var view = new ol.View({
            center: [0, 0],
            zoom: 2
        });

    // 创建第一个地图
  new ol.Map({
        layers: [
            new ol.layer.Tile({source: new ol.source.OSM()})
        ],
        view: view,
        target: 'map1'
  });

    // 创建第二个地图
    new ol.Map({
        layers: [
            new ol.layer.Tile({source: new ol.source.OSM()})
        ],
        view: view,
        target: 'map2'
  });
</script>

代码只有一点不同，即两个ol.Map使用了同一个view，就是这么神奇。
