OpenLayers 3 Primer

一个简单的地图

在介绍完ol3的基本信息之后，我们直接进入主题，下面我们看看用ol3做出来的Web地图是什么样的：

上面这个地图对应的代码如下：

<!Doctype html>
<html xmlns=http://www.w3.org/1999/xhtml>
<head>                  
    <meta http-equiv=Content-Type content="text/html;charset=utf-8">
    <meta http-equiv=X-UA-Compatible content="IE=edge,chrome=1">
    <meta content=always name=referrer>
    <title>OpenLayers 3地图示例</title>
    <link href="../ol3.13.1/ol.css" rel="stylesheet" type="text/css" />
    <script type="text/javascript" src="../src/ol3.13.1/ol.js" charset="utf-8"></script>
</head>

<body>
    <div id="map" style="width: 100%, height: 400px"></div>
    <script>
      // 新建一个地图
      new ol.Map({
            // 定义地图显示图层集合
            layers: [
              // 新建一个使用Open Street Map地图源的瓦片地图图层
              new ol.layer.Tile({source: new ol.source.OSM()})
            ],
            // 新建显示地图的视图
            view: new ol.View({
              center: [0, 0],    // 定义地图显示中心于经度0度，纬度0度处
              zoom: 2            // 并且定义地图显示层级为级别2
            }),
            // 设置id为map的dom元素来作为地图的容器
            target: 'map'    
        });
    </script>ch03
</body>

</html>

在html头部，我们引入了OpenLayers 3的库文件ol3.js和ol3.css，他们都可以在github上下载。为了在网页上显示地图，我们还需要一个用于存放地图的容器，此处是一个div元素，对应代码中id="map"的div。代码中<script>...</script>代码块中的是重头戏，地图就是这段代码创建的。可以看到有效代码只有7行，是不是非常的简单？！至于代码的含义，可以暂时参照代码中的注释来理解。看不懂也没关系，接下来将详细解读这段代码和地图结构。
